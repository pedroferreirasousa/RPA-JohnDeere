Autenticação Manual — JD Warranty System
1
Abra o JD Warranty System no seu navegador

Clique no botão abaixo. Faça o login normalmente (usuário, senha e aprovação no Okta Verify).

Abrir JD Warranty System
2
Abra as ferramentas de desenvolvedor (F12)

Após o login, pressione F12 → aba Network (Rede) → no filtro escreva products.
Aguarde um instante ou atualize a página — deve aparecer uma request para /api/products/....

3
Copie o Bearer token

Clique na request → aba Headers → procure Authorization em Request Headers.
Copie o valor inteiro: Bearer eyJ...

4
Cole o token abaixo e salve
